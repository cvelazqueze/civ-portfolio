Understanding this sample

This page documents a sub-application inside sample 32-graphql-federation-schema-first (part of a multi-service setup such as GraphQL federation). Each sub-app has its own main.ts and module tree.

Standalone HTTP application

This sample starts a normal web server. External clients reach it via URLs (browser, curl, mobile app).

When this shape is used: Best when you expose a REST/JSON API, MVC pages, Swagger, or file uploads from one deployable Node app.

HTTP server

Default port: 3002

Gateway — Federation Sample 32

Apollo Gateway composing schema-first subgraphs on :3000 (users) and :3001 (posts).

Quick start

Start both subgraphs first.

cd sample/32-graphql-federation-schema-first/gateway
npm install
npm run start:dev

GraphQL: http://localhost:3002/graphql
